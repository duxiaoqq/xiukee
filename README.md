# 有时候，心情就像一场毫无预兆的梅雨。

它不是那种轰轰烈烈的暴风雨，只是细细密密的、带着一层潮湿的雾气，把整颗心都浸得沉甸甸的。你坐在窗前，看着杯子里热气一点点散尽，说不上来哪里难过，但就是提不起劲。四周的声音好像都被隔绝在一层厚厚的玻璃之外，世界在运转，而你只想缩在自己的角落里，静静地和这点微小的低落相处。

但神奇的是，人心也像雨后的天空。

可能只是因为路边一朵刚开的小花，或是耳机里随机切到的一首轻快老歌，甚至只是风吹过来时恰到好处的温度，那一层厚重的云雾就会突然散开。一丝久违的金黄阳光探进心房，把那些潮湿的褶皱一点点熨平、晒干。

那一刻，心底会泛起微甜的酥麻感，像是有无数个粉色的气泡在轻轻撞击，整个人都变得轻盈起来。
